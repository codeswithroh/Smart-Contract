# WALLET SERVER

```
npm install
```

```
npm start
```

## GET API

```
url: http://127.0.0.1:3000/api/wallet?address=http://127.0.0.1:3000/api/wallet?address={address}

response: bool
```

## POST API

```
url: http://127.0.0.1:3000/api/wallet?address=http://127.0.0.1:3000/api/wallet

body: address: "address"
```
